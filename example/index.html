TweenSprite.js

Simple sprite animation plugin for TweenMax.js
Sequence animation (need some time to load)
TweenMax.sequence( domElement, sequenceList, 1.0, { delay: 0.5, repeat: -1 } );
Sprite Sheet animation by steps
TweenMax.spriteSheet( domElement, { 
	width: 818, 
	offsetX: 0,
	offsetY: 0,
	stepX: 201, 
	stepY: 154, 
	count: 11
}, 0.5, { delay: 0.1, repeat: -1 });
